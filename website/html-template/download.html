This page contains all the dumps of the OpenCitations Corpus (OCC) that are created regularly every month. Each dump is composed by several zip archives, each containing either data or provenance information of a particular sub-dataset of the OCC. After unzipping an archive, it is needed to use Disk ARchive (DAR) for recreating the whole structure - which is a multi-platform archive tool for managing huge amount of data.
